## 📌 About

CodeSunGrab is a premium command-line media downloader built with Python, wrapped in Node.js for npm distribution. Download videos & audio from multiple platforms with authentication, real-time progress tracking, and a beautiful Argentina-themed terminal interface.

---

## 📦 Install

```bash
npm install -g codesungrab
```

🚀 Run

```bash
codesungrab
# or short command
sungrab
```

✨ Features

```
🎥  Multi-Platform Download
    YouTube, TikTok, Instagram, Facebook, Twitter/X, Reddit,
    Twitch, Vimeo, SoundCloud, Dailymotion & more

🎵  Media Types
    • Video (MP4 - Best Quality)
    • Audio Only (MP3 - 192K)
    • Video + Subtitles (EN, BN, HI)

📋  Download Modes
    • Single URL
    • Batch (Multiple URLs - Sequential/Parallel)
    • Playlist (Full or Custom Range)
    • TikTok (With/Without Watermark)

🔐  Authentication System
    • Signup & Login
    • Email Verification (6-Digit Code)
    • Change Password
    • Update Profile
    • Session Management

🎨  Terminal UI
    • ASCII Art Banner
    • Typing Animations
    • Real-time Loading Spinners
    • Progress Bars
    • Argentina Flag Colors
    • User Profile Dashboard

⚙️  Advanced Settings
    • Proxy Support (HTTP/SOCKS5)
    • Speed Limit Control
    • Max Parallel Workers
    • Custom Download Path
    • Download History

🇦🇷  Argentina Victory Edition
    Dedicated to Argentina football spirit & Lionel Messi
```
